# VistaTab 🌟

一个简洁美观、功能丰富的Chrome浏览器新标签页扩展，为你提供更好的浏览体验。

![VistaTab](https://img.shields.io/badge/version-1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 特性

### 🔍 智能搜索
- **多搜索引擎支持**：Google、百度、必应等主流搜索引擎
- **搜索历史记录**：智能保存和管理搜索历史
- **键盘导航**：支持上下箭头键导航搜索历史
- **实时过滤**：输入时自动过滤匹配的历史记录
- **一键清除**：快速清除搜索历史

### 🚀 快捷网站管理
- **拖拽排序**：直观的拖拽操作重新排序网站
- **自动获取图标**：支持自动获取网站图标
- **智能布局**：根据网站数量自动调整布局
- **右键编辑**：快速编辑网站信息
- **响应式设计**：适配不同屏幕尺寸

### 🎨 主题定制
- **多主题支持**：浅色、深色、蓝色等多种主题
- **实时切换**：无需刷新即可切换主题
- **护眼设计**：深色主题减少眼部疲劳

### ⚙️ 个性化设置
- **灵活配置**：可自定义搜索引擎、网站行数等
- **数据同步**：设置自动保存到浏览器
- **隐私保护**：所有数据本地存储，保护隐私



## 🚀 安装

### 下载安装
1. 在GitHub的 [Releases](https://github.com/abcdream-Lary/VistaTab/releases) 页面下载最新版本的zip文件

2. 解压下载的zip文件到本地文件夹

3. 打开Chrome浏览器，进入扩展程序页面
```
chrome://extensions/
```

4. 开启右上角的"开发者模式"

5. 点击"加载已解压的扩展程序"

6. 选择解压后的VistaTab文件夹

7. 扩展程序安装完成！

### 开发者安装
如果你想参与开发或使用最新代码：

1. 克隆此仓库
```bash
git clone https://github.com/abcdream-Lary/VistaTab.git
```

2. 按照上述步骤3-7进行安装

## 🛠️ 开发

### 项目结构
```
VistaTab/
├── manifest.json          # 扩展程序清单
├── newtab.html            # 新标签页HTML
├── css/
│   └── newtab.css         # 样式文件
├── js/
│   └── newtab.js          # 主要逻辑
├── icons/                 # 图标文件
└── README.md
```

### 技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **Vanilla JavaScript** - 核心功能
- **Chrome Extension API** - 浏览器集成

### 本地开发
1. 克隆仓库
2. 在Chrome中加载扩展程序
3. 修改代码后刷新扩展程序即可看到效果
