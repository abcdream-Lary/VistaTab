# 更新日志

所有版本更改都将记录在此文件中。

## [1.2.0] - 2026-04-28

### 新增
- 🌤️ 新增天气小组件，显示在设置按钮左侧
- 🗂️ 全新设置面板，采用标签页导航设计
- 🎨 新增 4 种主题颜色（紫色、橙色、粉色）
- 🌙 **跟随浏览器/系统主题**功能，自动检测并应用主题
- 🔄 设置页面标签页分类：主题设置、天气设置、网站设置、数据管理
- 📊 存储容量监控，防止超出 Chrome 存储限制
- 🔄 网络请求错误处理优化，增加重试机制
- ⌨️ 搜索防抖优化，提升性能
- ℹ️ 设置面板底部新增"关于 VistaTab"区域

### 改进
- 🚀 优化搜索建议的防抖处理
- 🎨 优化设置面板 UI 交互体验
- 🐞 修复快捷网站在小窗口下显示不完整的问题
- 🐞 修复天气数据获取失败的问题
- 🎨 优化 Edge 浏览器主题兼容性

### 移除
- ❌ 移除壁纸功能（简化设置界面）
